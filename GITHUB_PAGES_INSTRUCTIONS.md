# 🚀 GitHub Pages Setup Instructions

## **Step 1: Login to GitHub**
1. **Go to:** https://github.com/login
2. **Enter your GitHub credentials**
3. **Sign in to access repository settings**

## **Step 2: Enable GitHub Pages**
1. **Navigate to:** https://github.com/wisserd/queoper/settings/pages
2. **Under "Source" section:**
   - Select "Deploy from a branch"
   - Choose branch: **main**
   - Choose folder: **/ (root)**
3. **Click "Save"**

## **Step 3: Access Your Live Site**
- **GitHub Pages URL:** https://wisserd.github.io/queoper/
- **Deployment time:** 2-5 minutes after enabling
- **Updates:** Automatic when repository changes
