# 📁 GitHub Repository Setup Commands

## After Creating Repository on GitHub:

### 1. Add Remote Origin
```bash
git remote add origin https://github.com/YOUR_USERNAME/OnPurpose.git
```

### 2. Set Main Branch
```bash
git branch -M main
```

### 3. Push to GitHub
```bash
git push -u origin main
```
