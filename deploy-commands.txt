# OnPurpose Deployment Commands

## Step 1: Create GitHub Repository
1. Go to: https://github.com/new
2. Repository name: OnPurpose
3. Make it Public
4. Click "Create repository"

## Step 2: Push Code to GitHub
# Replace YOUR_USERNAME with your GitHub username
git remote add origin https://github.com/YOUR_USERNAME/OnPurpose.git
git branch -M main
git push -u origin main

## Step 3: Railway Deployment
1. Go to: https://railway.app
2. New Project → Deploy from GitHub repo
3. Select OnPurpose repository
4. Add Service → Database → PostgreSQL

## Step 4: Environment Variables (Railway Variables tab)
NODE_ENV=production
PORT=3000
JWT_SECRET=your-32-character-secret-here
STRIPE_PUBLISHABLE_KEY=pk_live_...
STRIPE_SECRET_KEY=sk_live_...
STRIPE_WEBHOOK_SECRET=whsec_...
EMAIL_HOST=smtp.sendgrid.net
EMAIL_PORT=587
EMAIL_USER=apikey
EMAIL_PASS=your_sendgrid_api_key
EMAIL_FROM=noreply@yourdomain.com

## Step 5: Database Migration (Railway Run Command)
npx sequelize-cli db:migrate && npx sequelize-cli db:seed:all

## Step 6: Configure Stripe Webhook
URL: https://onpurpose.up.railway.app/api/payment/webhook
Events: payment_intent.succeeded, payment_intent.payment_failed

## Step 7: Test
node deploy/production-test.js https://onpurpose.up.railway.app
