### **Option 3: Create New Netlify Site**
1. Go to https://app.netlify.com/
2. Click "Add new site" → "Deploy manually"
3. Upload the deployment folder

### **Option 4: Zip and Upload**
1. Right-click `OnPurpose-Deploy` folder
2. "Send to" → "Compressed folder"
3. Upload the ZIP file to Netlify
